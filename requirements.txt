====================================================================
          AUTO-CAPTIONING TELEGRAM BOT REQUIREMENTS
====================================================================

[CORE FUNCTIONALITIES]
1. Bilingual Caption Generation
   - Auto-generate English/Bangla captions
   - Command: /language en|bn

2. Unlimited Unique Captions
   - Template-based generation system
   - 3 options per photo with random variants

3. Photo Context Detection
   - Supports: general, nature (expandable categories)

4. User Commands
   - /start : Initialize bot
   - /help  : Show instructions
   - /language en|bn : Set preference

[TECHNICAL SPECIFICATIONS]
1. Python 3.7+ (Compatible with 3.13)
2. Required Libraries:
   - python-telegram-bot>=20.0
   - logging (built-in)

3. File Structure:
   - bot.py : Main application
   - captions/ : Template directory (optional)

[SETUP INSTRUCTIONS]
1. Install dependencies:
   pip install python-telegram-bot --upgrade

2. Configure environment:
   - Get Telegram Bot Token from @BotFather
   - Replace 'YOUR_BOT_TOKEN' in bot.py

3. Execution:
   python bot.py
